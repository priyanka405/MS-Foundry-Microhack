# Student Guide

Welcome to the **MS Foundry MicroHack: Employee Support Agent**!

This guide helps you navigate the hackathon experience as a participant.

## What You'll Build

By the end of this MicroHack, you will have built a fully production-ready
**Employee Support Agent** using Microsoft Foundry — from a basic chat prompt
to an enterprise-ready, observed, and evaluated AI agent.

## How to Use This Guide

1. Start with `challenges/challenge0-setup/README.md`
2. Complete each challenge before moving to the next
3. Refer to the relevant `student-guide/challengeN.md` if you need extra hints
4. Only look at `solution-guide/` after completing a challenge yourself
