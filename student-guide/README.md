# Student Guide

Welcome to the **MS Foundry MicroHack: Employee Support Agent**.

This guide helps participants move through the full workshop from environment setup to enterprise-ready publishing.

## What You'll Build

By the end of this MicroHack, you will have built an **Employee Support Agent** using Microsoft Foundry that can:

- answer HR and IT support questions
- ground responses in trusted internal documents
- call tools such as a mock IT ticketing API
- remember useful employee context
- apply guardrails and safety controls
- be evaluated, optimized, observed, and prepared for rollout

## How to Use This Guide

1. Start with `challenges/challenge0-setup/README.md`.
2. Complete each numbered challenge in order.
3. Use the landing page in `docs/index.html` as your workshop home page.
4. Refer to `starter-code/` whenever a challenge points to provided assets.
5. Only look at `solution-guide/` after attempting a challenge yourself.

## Prerequisites

- Azure subscription with Foundry access
- Azure CLI and Bicep CLI installed
- Python 3.11+ or Node.js 20+
- VS Code with GitHub Copilot

## Workshop Path

- **Setup:** deploy the Azure environment with Bicep
- **Foundation:** choose a model and build the baseline agent
- **Capability:** add grounding, tools, and memory
- **Safety and quality:** apply guardrails and run evaluations
- **Operations:** optimize, observe, and publish
