# Starter Code

This directory contains boilerplate code to help participants get started with each challenge.

## Structure

```
starter-code/
├── documents/          # Synthetic HR and IT documents for grounding (Challenge 3)
│   ├── hr-vacation-policy.md
│   ├── hr-expense-policy.md
│   ├── it-password-reset.md
│   └── it-travel-policy.md
├── mock-api/           # Mock IT ticketing API (Challenge 4)
│   └── app.py
└── eval-dataset/       # Evaluation Q&A pairs (Challenge 7)
    └── eval-questions.json
```

## Usage

Each challenge's `README.md` references the relevant starter-code files.
Copy and adapt them as needed for your implementation.
