Hackathon overview

An AI teammate for the busiest people in your company

Executive assistants juggle 40+ meetings a week, thousands of documents, hundreds of open action items. This MicroHack shows you how to give them a real AI teammate — one that summarizes meetings, drafts follow-ups, searches enterprise knowledge, and triggers business workflows.

Learning outcomes

Objectives & prerequisites

What you'll learn, and what you need before you start.

🎯 Learning objectives

  • ✓ Build an agent in Microsoft Foundry (model + instructions + tools).
  • ✓ Ground the agent with enterprise knowledge and force citations.
  • ✓ Connect tools and actions to real business workflows.
  • ✓ Evaluate the agent for quality, safety, and groundedness.
  • ✓ Deploy and share the solution across Web, Teams, and API.

✅ Prerequisites

  • 🧠 Microsoft Foundry access (project on ai.azure.com).
  • ☁️ An Azure subscription or Foundry sandbox.
  • 📚 Basic understanding of what an agent is.
  • 📄 A few sample documents for grounding (notes, briefs, policies).
  • 🧰 Optional: VS Code and GitHub Copilot.
